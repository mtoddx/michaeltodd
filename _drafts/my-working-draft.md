This is a draft post! Because it is sitting in the `_drafts` folder, the live GitHub Pages website will **completely ignore it**. 

You can still preview it on your local machine if you run:
`bundle exec jekyll serve --drafts`

When you are finally ready to publish it to the world:
1. Move this file from the `_drafts` folder into the `_posts` folder.
2. Rename the file to include the current date at the start (for example: `2026-03-20-my-working-draft.md`).
