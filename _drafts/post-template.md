## Writing Tips

### SEO Best Practices
- Use descriptive titles (50-60 characters)
- Write compelling first paragraphs (they become excerpts automatically)
- Keep first paragraph concise (1-3 sentences, ~160 characters ideal)
- Use headings hierarchically (H1 → H2 → H3)
- Include relevant keywords naturally
- Add alt text to images
- Link to related posts

### Content Structure
- Start with why the topic matters
- Use short paragraphs (2-4 sentences)
- Break up text with headings
- Include examples and visuals
- End with actionable takeaways

### Markdown Tips
- Use `**bold**` for emphasis
- Use `*italic*` for subtle emphasis
- Use `` `code` `` for inline code
- Use `---` for horizontal rules
- Use `>` for blockquotes

### Before Publishing
- [ ] Proofread for typos
- [ ] Check all links work
- [ ] Verify math renders correctly
- [ ] Test code examples
- [ ] Add alt text to images
- [ ] Ensure first paragraph is compelling (it's your auto-excerpt!)
- [ ] Choose featured image (optional)
- [ ] Preview locally
- [ ] Run Lighthouse audit

### Publishing
1. Move from `_drafts/` to `_posts/`
2. Rename to `YYYY-MM-DD-title.md` format
3. Update the date in front matter
4. Commit and push
5. Verify on live site
6. Share on social media!
